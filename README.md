# Travel Router System 
> Projeto para estágio de TI na MMTech
--- 

# Funcionalidades
> - Criar viagens: O usuário pode criar viagens para organizar destinos dentro dela
> - Buscar destinos diferentes com mapa para orientar
> - Adicionar destinos na viagem 
> - Remover destinos da viagem
